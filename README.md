# Практична 8 - Функції вищого порядку
## Варіант 1, Data Transformation Library

## Що реалізовано

myMap, myFilter, myReduce - власні реалізації без вбудованих методів масиву

compose - композиція функцій справа наліво

pipe - композиція функцій зліва направо

curry - каррінг будь-якої функції з довільною кількістю аргументів

partial - часткове застосування, фіксує перші аргументи

memoize - кешує результати функції щоб не рахувати двічі

chain - chainable API для зручного ланцюжка викликів

## Запуск

node hof.js
